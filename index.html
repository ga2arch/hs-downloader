HDownload

Multipart threaded file downloader with counduits written in Haskell

View the Project on GitHub ga2arch/hs-downloader

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Features

  • Multipart threaded download
  • Use conduits to not abuse memory
  • Neat progressbar

Installation

git clone git@github.com:ga2arch/hs-downloader.git
cd hs-downloader
cabal install --prefix=$HOME --user

Now there will be an executable called hdownload in your home's bin.

Usage

hdownload <url>
